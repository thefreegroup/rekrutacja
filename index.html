Freedomes jest jednym z wiodących dostawców struktur eventowych
oraz konstrukcji tymczasowych i stałych na wielu rynkach zagranicznych.
W związku z dynamicznym rozwojem firmy rekrutujemy na stanowisko:

Programista PHP

Wymagania:

  • udział w tworzeniu nowych projektów i rozwiązań
  • praca z systemem kontroli wersji GIT
  • znajomość języka PHP5 na poziomie zaawansowanym
  • doświadczenie w programowaniu OOP oraz w strukturze MVC
  • tworzenie kodu zgodnie z dobrymi praktykami programowania
  • znajomość wzorców projektowych
  • wiedza z zakresu HTML5, CSS oraz jQuery
  • znajomość języka bazodanowego SQL
  • podstawowa znajomość środowiska Linux
  • język angielski na poziomie komunikatywnym

Mile widziane:

  • doświadczenie w korzystaniu oraz wytwarzaniu API
  • doświadczenie w pracy z dużymi bazami danych
  • obycie w tematyce cache i silników wyszukiwania
  • znajomość Symfony 2

Oferujemy:

  • pracę przy ciekawych projektach
  • atrakcyjne wynagrodzenie oraz bonusy finansowe
  • spokojną i przyjazną atmosferę pracy
  • zdobywanie cennego doświadczenia

Zadanie rekrutacyjne:

W ramach zadania rekrutacyjnego na stanowisko Programista PHP należy zaplanować (np. w formie graficznej) oraz wstępnie zrealizować projekt systemu służącego do skracania linków URL.

Założenia
  • system pracuje jako aplikacja webowa
  • dostęp do funkcjonalności skracania URL mają jedynie autoryzowani użytkownicy
  • adres URL tłumaczony jest na krótki, losowy ciąg znaków
  • możliwym ma być dodanie własnego ciągu znaków bez konieczności jego losowania
  • wywołanie skróconego adresu powinno zostać odnotowane

Projekt powinien zostać założony w osobnym repozytorium na GitHub jeszcze przed rozpoczęciem projektu. Na ocenę zadania rekrutacyjnego składa się również praca z repozytorium (commity, komentarze).
